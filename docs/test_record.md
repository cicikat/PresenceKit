## 评分标准
- 幻觉调用：角色说了查/做了某事，但工具实际没有触发 → 标记 ❌幻觉
- 误触发：不该查却触发了工具 → 标记 ❌误触发
- 漏触发：该查没有触发 → 标记 ❌漏触发
- 角色感：回复是否符合角色风格 → ✓正常 / △偏了 / ✗跑偏
